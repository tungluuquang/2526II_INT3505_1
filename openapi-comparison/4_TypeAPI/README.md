1. Chỉnh sửa và Thiết kế

Sử dụng TypeAPI Editor (như SDKgen Editor) để xem và chỉnh sửa file JSON này một cách trực quan.

    Import: Tải file library.typeapi.json vào Editor.

    Visual Edit: Thêm các thuộc tính hoặc endpoint mới mà không cần quan tâm đến cú pháp YAML/JSON phức tạp.

2. Sinh mã nguồn (SDK Generation)

Từ file định nghĩa này, bạn có thể sinh ra SDK cho nhiều ngôn ngữ khác nhau để sử dụng ở phía Client hoặc Server.

3. Việc tạo code được thực hiện trên sdkgen nên không có bước build
tạo testcase thì xuất phát là từ những file .ts.
